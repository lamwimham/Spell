# Spell 语音学习应用

## 项目简介
基于Expo的跨平台语音记录与拼写练习应用，支持Android/iOS/Web平台，采用Redux进行状态管理，支持SQLite本地存储。

## 开发环境搭建

### 1. 安装 Node.js
下载安装 [Node.js v18.x+](https://nodejs.org/zh-cn/)

### 2. 安装 Expo CLI
```bash
npm install -g expo-cli
```

### 3. 安装项目依赖
```bash
# 进入项目目录
cd /path/to/Spell

# 安装依赖
npm install
```

### 4. 启动开发服务器
```bash
npx expo start
```

## 项目运行
- 扫描二维码在Expo Go应用中运行
- 或使用 `a` 键启动Android模拟器
- 或使用 `i` 键启动iOS模拟器
